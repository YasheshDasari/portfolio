# Yashesh Dasari — Portfolio

Personal portfolio built with Next.js 15, Tailwind CSS v4, and next-themes.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- next-themes (dark/light mode)
- Lucide Icons
- Vercel (deployment)
